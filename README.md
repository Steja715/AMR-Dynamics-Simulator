# AMR Dynamics Simulator: Baddi Case Study
This repository contains a dynamic **SIR-based transmission model** designed to project the spread of antimicrobial resistance (AMR) in industrial hubs.

## 🔬 Research Context
The model is calibrated using original fieldwork data collected in **Baddi, Himachal Pradesh, India**. 
- **Sample Size:** 239 Pharmacies and 450 individuals.
- **Key Baseline:** Mean AMR Literacy (4.8) and Non-Prescription Sales (73%).

## 🧮 How it Works
The model utilizes Ordinary Differential Equations (ODEs) to track:
- **Sensitive Strains (Is):** Treatable infections.
- **Resistant Strains (Ir):** Infections driven by selection pressure from improper antibiotic use.
